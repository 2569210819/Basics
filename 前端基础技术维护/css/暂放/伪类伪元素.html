伪类 向某些选择器添加特殊的效果 (相当于 添加了一个实际的类 )

伪元素 将特殊的效果添加某些选择器里 (相当于 添加了一个实际的元素)

伪类 first-child ，:link:，vistited，:hover，:active，:focus，:lang

伪类分为三类 1，链接伪类 2，位置伪类

a元素获取焦点是tab键时选中的状态 链接伪类： :link {color: #FF0000} /* 未访问的链接 */ a:visited {color: #00FF00} /* 已访问的链接 */ a:hover {color: #FF00FF} /* 鼠标移动到链接上 */ a:active {color: #0000FF} /* 选定的链接 */
位置伪类： :first-child 伪类来选择元素的第一个子元素。 :last-child 选择元素的最后一个子元素 :nth-child(n) 选择元素的第 n 个元素，或符合某种公式的所有子元素 :nth-last-child(n) 选择元素的倒数第 n 个元素，或符合某种公式的所有子元素

伪元素有：:first-line，:first-letter，:before，:after

:first-letter 向文本的第一个字母添加特殊样式。 :first-line 向文本的首行添加特殊样式。 :before 在元素之前添加内容。 :after 在元素之后添加内容。
